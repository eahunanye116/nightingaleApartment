# **App Name**: Nightingale

## Core Features:

- Apartment Listings: Browse available apartments with details like size, amenities, and pricing.
- Search & Filter: Find apartments based on location, price range, number of guests, and desired amenities.
- Image Gallery: Display high-quality photos of each apartment to showcase its features.
- Booking Calendar: Check availability and book apartments for specific dates.
- AI-Powered Recommendations: Suggest apartments based on user preferences, booking history, and trending destinations, using a recommendation tool.
- Contact Host: Allow guests to ask their potential hosts more question using contact form

## Style Guidelines:

- Primary color: Saturated mustard (#D4A22D) evoking a sense of understated luxury.
- Background color: Desaturated warm cream (#F2F1E8).
- Accent color: Light terracotta (#E07A5F) to complement the primary mustard shade.
- Headline font: 'Playfair', a modern serif with an elegant feel.
- Body font: 'PT Sans', a sans-serif to ensure readability.
- Use clean and modern line icons for amenities and features.
- Clean and spacious layout with clear section divisions for easy navigation.
